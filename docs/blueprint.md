# **App Name**: SIWES AI Pro

## Core Features:

- Basic Info Input: Collects basic student and academic details like name, registration number, university, and department.
- Attachment Details Input: Gathers information about the place of attachment, supervisor, and company profile.
- Technical Details Input: Collects details on field of study, skills, technologies, and career path to personalize AI content generation.
- AI-Powered Company Profile: Generates a company profile using the company name, incorporating Google Search to augment its knowledge.
- AI Content Generation: Uses AI to generate acknowledgement, abstract, and challenges sections based on provided details, leveraging tool use and reasoning to synthesize diverse inputs.
- Live Preview: Provides a real-time preview of the report, updating as information is entered in the forms.
- AI Suggestion Tool: Provide contextual auto-suggestions for fields like skills, technology based on other fields such as Department.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey professionalism and trust.
- Background color: Light gray (#F9FAFB) to provide a clean, neutral backdrop.
- Accent color: Light blue (#3B82F6) for interactive elements and highlights, drawing user's eye and providing clear visual cues.
- Body and headline font: 'Inter', a versatile sans-serif for clear and modern readability.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, professional icons to represent different sections of the report.
- Subtle transition animations for a smooth user experience between steps and during content generation.
